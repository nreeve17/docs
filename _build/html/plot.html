biota.plot¶

plot_taxa¶

Plot the taxa summary of sample grouped by column

Parameters¶

data: pd.DataFrame
pandas dataframe contain feature by sample counts
sample_md: pd.Series
the metadata describing the samples to be plotted
feature_md: pd.Series
a series where the indices match the data tables columns the values are a category of interest, for example “Genus”
n_taxa: int, optional
the number of taxa to plot. This will plot the top n taxa in the data set default is 13
order: iterable
the order in which to plot the samples. by default will use the index of the get_tax_table() returned table.
ax: matplotlib.axes._subplots.AxesSubplot, optional
matplotlib axes object to plot on
color_series: pd.Series, optional
pandas series containing the colors that should be plotted. The index of the series should match the names of the taxa to be plotted. If unsure of what taxa is to be plotted use get_taxa_table to obtain the data. default is None. if None is passed a color map will be generated.
color_map: str, optional
string of the color map to be used. If continuous=True a continous map should be passed see http://matplotlib.org/examples/color/colormaps_reference.html for valid colormaps
legend: bool, optional
boolean whether to plot the legend. False will not plot the legend. Default is True
kind: str, optional
string of plot format to be produced. Default ‘bar’ will produce a bar plot
output_filename: str, optional
string of filename that the taxa dataframe will be written out to as a .csv file. If None, no file will be produced
**kwargs
Arbitrary keyword arguments.

Returns¶

ax: matplotlib.axes._subplots.AxesSubplot
The resulting axis object

Examples¶

>>> import pandas as pd
>>> feature_ids = list('efgh')
>>> sample_ids = list('abcd')

>>> data = pd.DataFrame([[0, 1, 1, 4],
...                      [1, 0, 3, 2],
...                      [1, 3, 0, 3],
...                      [4, 2, 3, 0]],
...                    index=sample_ids,
...                    columns=feature_ids)
>>> samples = pd.Series(list('xxyy'), index=sample_ids)
>>> features = pd.Series(['Taxa1', 'Taxa1', 'Taxa3','Taxa2'],
                         index=feature_ids)
>>> feature_colors = pd.Series(['#66c2a5','#fc8d62','#8da0cb'],
                               index=['Taxa1', 'Taxa2','Taxa3'])
>>> plot_taxa(data, samples, features, color_series=feature_colors)
