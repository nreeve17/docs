biota.util¶

rarefy¶

Subsample the rows of a table of counts. The resulting counts for each of the rows will sum to the depth. The random sampling happend without replacement.

Parameters¶

table : pd.DataFrame
A pandas dataframe containing counts (numeric whole numbers)
depth : int
Depth at which to subsample to

Returns¶

results_table : pd.DataFrame
A pandas DataFrame where the sum of each row is equal to the rarefaction depth. Rows that sum to less than the depth are not included in the final table

Examples¶

>>> table = pd.DataFrame([[5, 6], [10, 11]])
>>> rarefy(table, 5)
  0 1
0 3 2
1 2 3

get_dms¶

Function to create distance matrices from a sample by feature table.

Parameters¶

data: pd.DataFrame
The sample by feature count data
rooted_tree: qiime2.sdk.Results
A QIIME2 Artifact containing the midpoint rooted tree
metrics : iterable of str
An iterable of metrics to calculate the distance matricies by. any qiime metrics are valid, tree is required for unifrac distances.
sampling_depth : int
The depth at which to rarefy the data table
output_dir : str
The name of the output directory where to store the output files
overwrite: bool
If True new distance matrices will be created regardless if they exist or not
n_jobs: int
The number of jobs to start for diversity methods (excluding weighted_unifrac).

Returns¶

dms : dict
A dictionary containing the distance matricies. the keys are the names of the metrics that were calculated.

Example¶

>>> tree = biota.util.make_tree(features.DenoisedSequenceVariant,
...                             overwrite=False)
>>> dms = biota.util.get_dms(data, sampling_depth=5000, rooted_tree=tree)
