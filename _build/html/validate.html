Biota Metadata API¶

fetch_google_sheet¶

Fetches a remote metadata sheet from Google Sheets

Parameters¶

spreadsheet_id: str
The hash from a Google Sheet URL where https://docs.google.com/spreadsheets/d/1lMTqWFvy2pgJctK3dz2jT5ER2DBjHd671yapIDMWgmI is “1lMTqWFvy2pgJct K3dz2jT5ER2DBjHd671yapIDMWgmI”
range_name: str
The Sheet (and optionally subset) of the Google Sheet to retrieve
**kwargs
Keyword arguments to be passed to pd.read_csv

Returns¶

pd.DataFrame
The Google Sheet as a pandas DataFrame object

Example¶

>>> df = fetch_google_sheet(spreadsheet_id,
...                         range_name,
...                         skiprows=1,
...                         index_col=0)
